# Shopix

A modern responsive e-commerce web application built with React, Redux Toolkit, and Tailwind CSS.

## Project Status

🚧 **Work in Progress**

This project is currently under development. I am actively working on new features, UI improvements, and better responsiveness. More updates will be added soon.

## Current Features

- Responsive navigation bar
- Product cards
- Product details page
- Category section
- Cart functionality
- Quantity controls
- Checkout page UI
- Redux state management

## Upcoming Improvements

- Better authentication flow
- Performance optimization
